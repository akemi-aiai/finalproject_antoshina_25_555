# Настройка API для Parser Service
## 1. ExchangeRate-API (Фиатные валюты)
### Регистрация:
1. Перейди на https://www.exchangerate-api.com/
2. Нажми "Get Free Key" 
3. Зарегистрируйся с email
4. Получи API ключ (выглядит как `3b47a9b92e1b14c1f1234567`)

### Настройка:
Добавь ключ в переменные окружения:
```bash
export EXCHANGERATE_API_KEY=37ad3540dcf011478170ac93
